PROJECT IMAGES — how to add them
=================================

Each project shows 8 thumbnails. Drop your images in THIS folder
(clone/projimg/) using the exact filenames below. Recommended: square-ish
JPGs, ~800x800px or larger. PNG also works if you rename the extension to
.jpg, or tell me and I'll switch the code to .png.

Filename pattern:  p<ProjectNumber>-<ImageNumber>.jpg   (image 1 to 8)

Project 1  — Gearbox Assembly ................... p1-1.jpg ... p1-8.jpg
Project 2  — Lightweight Bicycle Frame .......... p2-1.jpg ... p2-8.jpg
Project 3  — Parametric Brake Disc Rotor ........ p3-1.jpg ... p3-8.jpg
Project 4  — Sheet Metal Electronic Enclosure ... p4-1.jpg ... p4-8.jpg
Project 5  — Additive Manufacturing Gripper ..... p5-1.jpg ... p5-8.jpg
Project 6  — DELMIA Assembly Process Plan ....... p6-1.jpg ... p6-8.jpg
Project 7  — Electric Motor Workstation Line .... p7-1.jpg ... p7-8.jpg
Project 8  — CNC Machining Process Setup ........ p8-1.jpg ... p8-8.jpg
Project 9  — ENOVIA Enterprise PLM .............. p9-1.jpg ... p9-8.jpg
Project 10 — AEC Parametric Architectural ....... p10-1.jpg ... p10-8.jpg

Notes
-----
- Until a file exists, its thumbnail shows a numbered placeholder and the
  popup shows "This image hasn't been added yet." Nothing looks broken.
- Click any thumbnail to open the popup, then use the on-screen arrows or
  your keyboard Left/Right arrow keys to move through that project's 8
  images. Esc or click outside to close.
- You don't need all 8 — add as many as you have; the rest stay as
  placeholders.
